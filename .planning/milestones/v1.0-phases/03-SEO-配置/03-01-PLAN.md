<task type="auto">
  <name>Task 1: 更新 hugo.yaml 添加 SEO 配置</name>
  <files>hugo.yaml</files>
  <action>
    在 hugo.yaml 中添加以下配置：

    1. `enableRobotsTXT: true` - 启用自动生成 robots.txt
    2. 添加 `sitemap` 配置项：
       - changefreq: "weekly"
       - priority: 0.5
       - filename: "sitemap.xml"
    3. 添加 `disableKinds` 排除不需要的页面类型：
       - taxonomy, term, RSS, sitemap
    4. 在 `params` 中添加 SEO 相关配置：
       - description: 全站描述
       - keywords: 全站关键词数组
       - social.twitter: Twitter 账号
       - assets.favicon: favicon 配置

    参考 RESEARCH.md 中的完整配置示例。
  </action>
  <verify>
    <automated>hugo -E --quiet 2>&1 | head -5 && test -f public/robots.txt && test -f public/sitemap.xml</automated>
  </verify>
  <done>hugo.yaml 包含 enableRobotsTXT、sitemap、params SEO 配置项</done>
</task>

<task type="auto">
  <name>Task 2: 更新示例文章 front matter 添加 keywords 和 cover</name>
  <files>content/tech/hello-world.md, content/fiction/first-story.md</files>
  <action>
    为每篇文章的 front matter 添加以下字段：

    1. keywords: 关键词数组（如 ["Hugo", "PaperMod", "博客"]）
    2. cover: 封面图配置（用于 Open Graph 和 Twitter Card）
       - image: 封面图路径（使用占位符或外部图片 URL）
       - alt: 封面图描述

    注意：由于当前没有本地图片，可使用外部占位图（如 picsum.photos）或在后续添加本地图片。

    保持现有的 description 字段不变（已在 hello-world.md 中存在）。
  </action>
  <verify>
    <automated>hugo -E && grep -q "keywords" public/tech/hello-world/index.html && grep -q "og:image" public/tech/hello-world/index.html</automated>
  </verify>
  <done>每篇文章包含 keywords 和 cover 配置</done>
</task>

<task type="auto">
  <name>Task 3: 创建自定义 robots.txt 模板支持百度爬虫</name>
  <files>layouts/robots.txt</files>
  <action>
    在 layouts/robots.txt 创建自定义模板：

    1. 创建 layouts 目录（如果不存在）
    2. 创建 layouts/robots.txt 文件，内容包括：
       - User-agent: * 允许所有爬虫
       - User-agent: BaiduSpider 明确允许百度爬虫
       - Sitemap 指向 sitemap.xml
       - Disallow: /search/ 等私有路径

    使用 Hugo 模板语法：`{{ "sitemap.xml" | absURL }}` 生成绝对 URL。

    注意：PaperMod 主题默认会生成只允许 Google 的 robots.txt，自定义模板可以覆盖默认行为以支持百度。
  </action>
  <verify>
    <automated>test -f layouts/robots.txt && hugo -E && grep -q "BaiduSpider" public/robots.txt</automated>
  </verify>
  <done>public/robots.txt 包含百度爬虫允许规则</done>
</task>
