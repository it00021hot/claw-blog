<task type="auto">
  <name>Task 2: 实现 GitHub Fetcher 和 Blog Fetcher</name>
  <files>scripts/fetchers/__init__.py, scripts/fetchers/github_fetcher.py, scripts/fetchers/blog_fetcher.py</files>
  <action>
实现两个 fetcher 类：

1. 创建 scripts/fetchers/github_fetcher.py：
   - 继承 BaseFetcher
   - fetch_issues(owner, repo, token=None): 使用 GitHub API 抓取 Issues
     - 请求头: Accept: application/vnd.github+json, X-GitHub-Api-Version: 2022-11-28
     - 如有 token 添加 Authorization: Bearer {token}
     - 参数: state=all, per_page=100, sort=updated
     - 返回: List[Article]
   - fetch_readme(owner, repo, token=None): 抓取 README
     - 使用 base64 解码内容
     - 返回: Article 或 None
   - can_fetch(url): 检查是否为 github.com 域名

2. 创建 scripts/fetchers/blog_fetcher.py：
   - 继承 BaseFetcher
   - fetch_article(url): 使用 httpx + BeautifulSoup4 抓取
     - headers: User-Agent: Mozilla/5.0 (避免被拦截)
     - timeout: 30 秒
     - 提取标题: soup.find('h1') 或 <title>
     - 提取内容: 移除 script/style/nav/footer，提取 article 或 main 或 body
     - 返回: Article
   - can_fetch(url): 检查是否为博客域名

3. 更新 scripts/fetchers/__init__.py：导出 fetcher 类

注意：处理异常时返回空列表或记录日志，不要中断执行
  </action>
