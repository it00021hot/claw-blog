<name>Task 5: 创建 GitHub Actions 工作流</name>
<files>.github/workflows/fetch-articles.yaml</files>
<action>
创建 GitHub Actions 工作流文件：

1. 创建 .github/workflows/fetch-articles.yaml：
   - name: Fetch Articles
   - on:
     - schedule: cron "0 8 * * *" (每天 8:00 UTC)
     - workflow_dispatch: 手动触发
   - permissions: contents: read
   - jobs:
     - fetch:
       - runs-on: ubuntu-latest
       - steps:
         - name: Checkout
           uses: actions/checkout@v4
         - name: Set up Python
           uses: actions/setup-python@v5
           with:
             python-version: '3.11'
         - name: Install dependencies
           run: |
             pip install -r requirements-fetch.txt
           env:
             OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
             GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
         - name: Run fetch script
           run: python scripts/main.py
         - name: Commit new articles
           if: github.event_name != 'workflow_dispatch'
           run: |
             git config --local user.email "github-actions[bot]@users.noreply.github.com"
             git config --local user.name "github-actions[bot]"
             git add content/tech/ content/fiction/
             git diff --staged --quiet || git commit -m "chore: add fetched articles"
             git push

2. 创建 requirements-fetch.txt：
   httpx>=0.28.1
   beautifulsoup4>=4.14.3
   openai>=1.0.0
   python-dotenv>=1.0.0
   python-dateutil>=2.8.0

注意：
- 使用 secrets 存储 OPENAI_API_KEY 和 GITHUB_TOKEN
- workflow_dispatch 允许手动触发测试
- 自动 commit 仅在定时任务时执行
